# img-to-ascii

## Description

Most ascii image converters use a simple brightness range from ' ' to '@' for every character-sized region.
This is a fast approach, but not as precise.
This script instead iterates through every ASCII character in every region to find the character that best matches the actual bright and dark sections of that region.
Uses OpenCV to overlay images of different characters and measure the fit to the region of interest.

## Execution

Runs from command line with

`python3 img_to_ascii.py <image-path> <output-line-width>`
